# AI Grievance Redressal Agent for Government Schemes

A Flask-based web application that helps Indian citizens check eligibility and apply for government schemes like PM-Kisan, Ration Card, and Pension.

## Features

- Multilingual support (English, Hindi, Telugu)
- Voice input using Web Speech API
- Scheme detection using ML (TF-IDF + Naive Bayes)
- Rule-based eligibility checking
- Step-by-step application guidance
- Government-branded UI with Bootstrap 5
